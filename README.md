# 坦克大战 (Tank Battle)

一个使用 HTML5 Canvas + JavaScript 实现的经典坦克大战游戏。

## 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` | 发射炮弹 |

## 游戏规则

- 控制绿色坦克消灭红色敌方坦克
- 每消灭一辆敌方坦克获得 100 分
- 基地（底部黄色区域）被击中则游戏结束
- 生命耗尽则游戏结束

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6 Class)
- 无外部依赖
